git add: Este comando se usa para añadir cambios de archivos al área de preparación (staging area) antes de hacer un commit. Básicamente, le dices a Git qué archivos quieres incluir en el próximo commit.


git commit -m "mensaje": Este comando guarda los cambios añadidos en el área de preparación en el repositorio y les asigna un mensaje descriptivo. El -m "mensaje" es para incluir una nota que describa los cambios realizados.
